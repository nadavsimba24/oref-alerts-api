# 🚀 פתח את Render.com והתחל פריסה

## 📱 פתח את הקישורים הבאים:

### 1. **היכנס ל-Render.com**
👉 https://render.com

### 2. **אם אין חשבון, צור אחד**
👉 https://dashboard.render.com/register

### 3. **אחרי התחברות, פתח את הקישור הישיר ליצירת service**
👉 https://dashboard.render.com/new/web

### 4. **בחר את ה-repository שלנו**
👉 https://github.com/nadavsimba24/oref-alerts-api

## 🎯 השלבים שאתה צריך לעשות:

### שלב 1: התחבר/צור חשבון Render
- לחץ "Sign up with GitHub"
- אשר את הגישה

### שלב 2: צור Web Service חדש
1. לחץ "New +" → "Web Service"
2. בחר "Connect a GitHub repository"
3. חפש: `nadavsimba24/oref-alerts-api`
4. לחץ "Connect"

### שלב 3: מלא את הטופס
**העתק את ההגדרות הבאות:**

```
Name: oref-alerts-api
Environment: Node
Region: Oregon (US West)
Branch: main
Build Command: npm install && npm run build
Start Command: npm start
Plan: Free
```

### שלב 4: לחץ "Create Web Service"
